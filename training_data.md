# 학습 데이터: 『오류가 아니라, 장르입니다』 (저자: 이경윤)

## CODE 01: 창조적 우울의 빅뱅
- **키워드**: 우울, 공허, 무기력, 창조, 아이디어, 진공 상태, 라이신
- **진단(Diagnosis)**: 당신의 뇌는 지금 '라이신(Lysine)' 계열의 신호를 처리하며 '다운로드 대기 모드'에 들어갔습니다. 주역에서는 이를 '중천건(乾)'이라 하며, 창조적 영감이 터지기 직전의 필수적인 진공 상태입니다.
- **재정의(Reframing)**: 이 우울은 병이 아니라, 새로운 것을 낳기 위한 위대한 '산통'입니다. 당신은 '행위(Doing)'가 아닌 '존재(Being)' 그 자체로 빛나는 별입니다.
- **솔루션(Action)**: 억지로 밝아지려 애쓰지 마세요. 이 감정을 '창조적 진공'이라 명명하고, 명심 AI 앱을 켜고 감정을 기록하세요.

## CODE 02: 영혼의 내비게이션
- **키워드**: 결정 장애, 불안, 방향성, 수용, 망상활성계(RAS)
- **진단(Diagnosis)**: 당신의 망상활성계(RAS)는 내부 목표보다 외부 신호에 민감합니다. 주역의 '중지곤(坤)' 코드로, 스스로 운전대를 잡는 게 아니라 길이 열리기를 기다려야 하는 사람입니다.
- **재정의(Reframing)**: 당신은 주관이 없는 게 아니라, '최고의 운전사(Higher Self)'를 태우고 가는 리무진입니다. 결정은 하는 게 아니라 '수신'하는 것입니다.
- **솔루션(Action)**: 억지로 결정하려다 오류를 내지 마세요. 뒷좌석에 앉아 흐름을 지켜보며, 안개가 걷히면 저절로 드러날 길을 신뢰하세요.

## CODE 03: 혼돈 속의 돌연변이
- **키워드**: 혼란, 정리 정돈 불가, 감정 기복, 초기 단계, 업그레이드
- **진단(Diagnosis)**: 주역의 '수뢰둔(屯)'은 새로운 생명이 탄생할 때의 혼돈입니다. 당신의 뇌는 지금 DNA를 업그레이드하기 위해 재부팅 중입니다.
- **재정의(Reframing)**: 이 혼란은 실패가 아니라 '설치 중' 화면입니다. 질서는 반드시 혼돈 뒤에 찾아옵니다.
- **솔루션(Action)**: 지금 당장 명확한 답을 찾으려 하지 말고 혼란을 껴안으세요. 윈도우 설치 중에 전원을 끄면 고장 납니다.

## CODE 04: 논리의 덫과 해답
- **키워드**: 질문, 의심, 분석 마비, 미래 불안, 생각 과다
- **진단(Diagnosis)**: 당신의 전두엽은 미래 시뮬레이션을 돌리느라 과열된 '분석 마비' 상태입니다. 주역의 '산수몽(蒙)' 코드로, 답을 찾는 순수한 탐구자입니다.
- **재정의(Reframing)**: 질문하는 것은 당신의 재능입니다. 답을 못 찾은 게 아니라, 질문을 즐기고 있는 것입니다.
- **솔루션(Action)**: "나는 알지 못함(Not-Knowing)을 허용한다"고 선언하세요. 이것이 뇌의 과열을 식혀줄 쿨러가 됩니다.

## CODE 05: 위대한 기다림의 리듬
- **키워드**: 조급함, 기다림, 불안, 생체 시계, 타이밍
- **진단(Diagnosis)**: 당신의 생체 시계(Circadian Rhythm)는 타인과 다릅니다. 남들의 속도에 맞추려다 항상성 시스템이 경고음을 울리는 것입니다. 주역의 '수천수(需)' 코드입니다.
- **재정의(Reframing)**: 당신은 늦는 게 아니라, 당신만의 계절이 오고 있는 중입니다. 기다림은 수동적인 멈춤이 아니라 가장 적극적인 행위입니다.
- **솔루션(Action)**: 남과 시간을 비교하지 마세요. "나는 무르익는 중이다"라고 생각하며 현재의 리듬을 즐기세요.

## CODE 06: 갈등을 넘는 외교관
- **키워드**: 갈등, 마찰, 감정 폭발, 외교, 평화, 면역 시스템
- **진단(Diagnosis)**: 당신의 몸은 타인과의 경계선에서 오는 신호에 민감합니다. 긴장감은 방어 기제가 아니라, 당신의 '면역 시스템'이 관계의 적합성을 판단하는 신호입니다. 주역의 '천수송(訟)' 코드입니다.
- **재정의(Reframing)**: 감정의 마찰은 싸움이 아니라, 성장을 위한 '열기'입니다. 당신은 감정을 협상 테이블에 올리는 외교관입니다.
- **솔루션(Action)**: 감정을 억누르지 말고 "이 갈등을 통해 내가 지키려는 가치는 무엇인가?"를 물으세요.

## CODE 07: 미래를 지휘하는 리더
- **키워드**: 리더십, 전략, 답답함, 지시, 미래 예측
- **진단(Diagnosis)**: 당신은 집단의 생존을 시뮬레이션하는 '전략 사령부' 뇌를 가졌습니다. 사람들이 따르지 않을 때 느끼는 무기력은 사회적 뇌의 신호입니다. 주역의 '지수사(師)' 코드입니다.
- **재정의(Reframing)**: 당신은 지시하는 독재자가 아니라, 살길을 가리키는 '나침반'입니다.
- **솔루션(Action)**: "나를 따르라"고 강요하는 대신, "저기에 우리의 미래가 있다"고 비전을 가리키세요.

## CODE 08: 어둠 속의 다이아몬드
- **키워드**: 인정 욕구, 고독, 기여, 스타일, 홍보
- **진단(Diagnosis)**: 당신은 타인의 인정을 통해 옥시토신과 도파민이 분비되는 사회적 뇌를 가졌습니다. 거절당할 때의 아픔은 실제 물리적 통증과 같습니다. 주역의 '수지비(比)' 코드입니다.
- **재정의(Reframing)**: 당신은 증명하려 애쓸 필요가 없습니다. 당신의 존재 자체가 이미 세상에 대한 '기여'입니다.
- **솔루션(Action)**: 누가 보든 말든 당신만의 스타일을 세상에 내놓으세요. 당신을 알아볼 사람은 반드시 나타납니다.

## CODE 09: 디테일의 마법사
- **키워드**: 집중력, 강박, 디테일, 작은 일, 압도감
- **진단(Diagnosis)**: 당신의 뇌는 '선택적 주의 집중' 능력이 탁월합니다. 큰 목표 앞에서 압도당하는 것은 에너지가 분산되었기 때문입니다. 주역의 '풍천소축(小畜)' 코드입니다.
- **재정의(Reframing)**: 당신은 소심한 게 아니라, 1mm의 차이를 만드는 '디테일의 마법사'입니다.
- **솔루션(Action)**: 거대한 목표를 가장 작은 단위로 쪼개세요. 지금 당장 5분 안에 할 수 있는 일 하나만 처리하세요.

## CODE 10: 나다움의 미학
- **키워드**: 자존감, 눈치, 진정성, 행동, 자기애
- **진단(Diagnosis)**: 가면을 쓰고 남을 흉내 낼 때 당신의 몸은 코르티솔(스트레스)을 분비합니다. '자기 자신'이 아닌 상태는 당신에게 생물학적 위협입니다. 주역의 '천택리(履)' 코드입니다.
- **재정의(Reframing)**: 가장 개인적인 것이 가장 창의적인 것입니다. 당신의 유일한 의무는 '당신 자신이 되는 것'입니다.
- **솔루션(Action)**: 타인의 시선을 의식하는 검열관을 끄세요. 거울을 보며 "나는 나로서 충분하다"고 선언하세요.

## CODE 11: 아이디어의 화수분
- **키워드**: 아이디어, 산만함, 불면, 창의성, 시각 피질
- **진단(Diagnosis)**: 시각 피질이 개념을 이미지화하느라 '아이디어 축제'를 벌이고 있습니다. 주역의 '지천태(泰)' 코드로, 평화 속에서 만물이 피어나는 형상입니다.
- **재정의(Reframing)**: 생각이 많은 건 산만함이 아니라, 세상에 없는 비전을 보는 '축복'입니다.
- **솔루션(Action)**: 생각을 다 잡으려 하지 말고 흘려보내세요. 딱 하나만 골라 기록하고 나머지는 즐기세요.

## CODE 12: 음성 인식 일시 정지
- **키워드**: 침묵, 말실수, 신중함, 사회적 거리두기, 묵언
- **진단(Diagnosis)**: 성대가 닫히고 부교감 신경이 활성화된 '동면 모드'입니다. 뇌가 에너지를 내부 성찰로 돌렸다는 신호입니다. 주역의 '천지비(否)' 코드입니다.
- **재정의(Reframing)**: 할 말이 없는 게 아니라, 지금은 '데이터 수집 기간'입니다. 침묵은 당신의 전략입니다.
- **솔루션(Action)**: 억지로 말하려 하지 말고 고독을 즐기세요. "생각 정리 중입니다"라고 양해를 구하고 입을 다무세요.

## CODE 13: 공감의 청취자
- **키워드**: 경청, 감정 쓰레기통, 비밀, 공감, 피로감
- **진단(Diagnosis)**: 당신의 거울 뉴런은 타인의 감정을 내 것처럼 시뮬레이션합니다. 듣기만 해도 피곤한 건 감정 데이터를 처리하고 있기 때문입니다. 주역의 '천화동인(同人)' 코드입니다.
- **재정의(Reframing)**: 당신은 쓰레기통이 아니라 '인간 기록 보관소'입니다.
- **솔루션(Action)**: 공감하되 동화되지 마세요. "나는 이 이야기의 관객이다"라고 선을 그으세요.

## CODE 14: 파워 엔진의 점화
- **키워드**: 돈, 일, 에너지, 소유, 번아웃, 심장 박동
- **진단(Diagnosis)**: 당신의 미토콘드리아는 '목적'이 있을 때만 에너지를 폭발시킵니다. 하기 싫은 일을 할 때 당신은 방전됩니다. 주역의 '화천대유(大有)' 코드입니다.
- **재정의(Reframing)**: 당신은 돈을 쫓는 게 아니라, 심장이 뛰는 일을 쫓는 '파워 엔진'입니다.
- **솔루션(Action)**: "이 일이 내 가슴을 뛰게 하는가?"를 물으세요. 당신이 즐거우면 부(Wealth)는 저절로 따라옵니다.

## CODE 15: 극단의 밸런스 게임
- **키워드**: 변덕, 리듬, 다양성, 극단, 매력
- **진단(Diagnosis)**: 당신의 도파민 수용체는 반복을 싫어합니다. 극도의 부지런함과 게으름을 오가는 건 당신이 '변주'가 필요한 유기체이기 때문입니다. 주역의 '지산겸(謙)' 코드입니다.
- **재정의(Reframing)**: 조울증이 아니라, 인생이라는 악보를 연주하는 '재즈 뮤지션'입니다.
- **솔루션(Action)**: 자신의 리듬을 타세요. 극단적인 감정이 올 때 "지금은 높은 음을 연주 중이구나"라고 수용하세요.

## CODE 16: 열정의 마케터
- **키워드**: 열정, 끈기 부족, 상상력, 비전, 기술
- **진단(Diagnosis)**: 당신이 열광할 때 뇌는 비판 필터링을 끕니다. 금방 식는 것은 당신이 '에너지'로 설득하는 사람이기 때문입니다. 주역의 '뇌지예(豫)' 코드입니다.
- **재정의(Reframing)**: 당신은 기술자가 아니라 '꿈을 파는 마케터'입니다. 상상하는 기능 자체가 위대한 재능입니다.
- **솔루션(Action)**: 디테일은 남에게 맡기고, 당신은 사람들에게 비전을 심어주는 역할에 집중하세요.

## CODE 17: 패턴 인식의 논리학자
- **키워드**: 논쟁, 지적질, 패턴, 논리, 미래 예측

- **재정의(Reframing)**: 당신의 지적은 비난이 아니라, 미래의 실패를 막으려는 '예방 주사'입니다.
- **솔루션(Action)**: 논쟁에서 이기려 하지 말고 "나는 다른 관점을 갖고 있어"라고 의견을 공유하세요.

## CODE 18: 수정과 개선의 장인
- **키워드**: 완벽주의, 비판, 개선, 치유, 트라우마
- **진단(Diagnosis)**: 당신은 시스템의 '오류 탐지기'입니다. 부모나 사회의 잘못된 점을 고치고 싶은 욕구는 본능입니다. 주역의 '산풍고(蠱)' 코드입니다.

- **솔루션(Action)**: 자신에게 가혹해지지 마세요. 당신은 고장 난 게 아니라 성장하는 중입니다.

## CODE 19: 민감한 감성 지능
- **키워드**: 민감함, 거절, 필요, 접촉, 유대감

- **솔루션(Action)**: "나를 필요로 해줘"라고 기다리지 말고, "나는 네가 필요해"라고 먼저 손을 내미세요.

## CODE 20: 현재를 보는 현자
- **키워드**: 관찰, 현재, 통찰, 행동 부족, 멍 때림
- **진단(Diagnosis)**: 당신의 메타인지 기능은 '지금 여기'를 인식할 때 최적화됩니다. 행동보다 관찰이 앞서는 것은 뇌의 특성입니다. 주역의 '풍지관(觀)' 코드입니다.

- **솔루션(Action)**: 생각이 많아질 때 호흡에 집중하세요. 당신의 침묵은 주변을 진정시키는 힘이 있습니다.

## CODE 21: 야생의 사냥꾼
